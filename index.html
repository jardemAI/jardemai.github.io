Why open?

If this closes, the region loses infrastructure.

Järdem AI is not only an app. It is meant to become reusable open infrastructure for Central Asian developers, NGOs, educators, civic-tech teams, and public-interest AI builders.

Open codeAssistant interface, retrieval pipeline, safety logic, and deployment tools.
Open datasetsPublic-service questions, scam examples, and document simplification tasks where legally possible.
Open evaluationsBenchmarks for Kazakh/Russian mixed-language understanding and practical local use cases.
Roadmap

From demo to open local AI stack.

Phase 1

Public demo

Landing page, interactive scenarios, GitHub repository, early prompt logic, and Kazakhstan-focused use cases.

Phase 2

Open prototype

Web assistant, scam-check module, document explanation module, public-service navigation, and small evaluation dataset.

Phase 3

Local-first AI

Lightweight model experiments, offline selected tasks, mixed-language benchmark, and community contributions.

Phase 4

Central Asia expansion

Uzbek and Kyrgyz support, NGO partnerships, more public datasets, developer tools, and documentation.
